# <a name="environment-options-dialog-box"></a>Środowisko, opcje — Okno dialogowe
Możesz użyć stron w **środowiska** folderu w **opcje** okno dialogowe, aby ustawić sposób niektóre elementy zintegrowanego środowiska programistycznego (IDE) wyświetlane i się zachowują. Możesz uzyskać dostęp do stron środowisko, na pasku menu, wybierając **narzędzia**, **opcje**, a następnie wybierając **środowiska**.

> [!NOTE]
> Okna dialogowe i polecenia menu, które są wyświetlane na komputerze, mogą się różnić od tych, które są opisane w pomocy, w zależności od aktywnych ustawień i używanej wersji Visual Studio. Aby zmienić swoje ustawienia, wybierz opcję **Import i eksport ustawień** na **narzędzia** menu. Aby uzyskać więcej informacji, zobacz [personalizowanie środowiska IDE programu Visual Studio](../../ide/personalizing-the-visual-studio-ide.md).
